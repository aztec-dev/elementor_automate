How It Works

  1. Write your blog post in Markdown with YAML frontmatter (title, categories, tags, SEO metadata)
  2. Upload images that you reference in the markdown
  3. Submit the form with your WordPress credentials
  4. The app will:
    • Upload all images to your WordPress media library
    • Convert markdown to HTML
    • Create categories and tags if they don't exist
    • Set featured image, SEO metadata (Yoast compatible)
    • Optionally enable Elementor page builder support
    • Create the post as draft or publish immediately
